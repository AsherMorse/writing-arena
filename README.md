# AI-Powered Competitive Writing Platform

## Project Overview

This platform transforms writing instruction through extrinsic motivation and competitive gameplay. Students participate in Counterstrike-style writing matches, earning points, ranks, and achievements while developing mastery across five writing traits.

- **Competitive Matchmaking**: Counterstrike-style party formation with AI players
- **Balanced Motivation System**: Intrinsic motivation support with strategic extrinsic rewards
- **Capstone Project**: Op-ed writing reviewed by editors and published in local newspapers
- **Formative Assessment**: AI-driven learning guidance and mastery tracking

## Architecture

- **Frontend**: TypeScript/React/Next.js with real-time WebSocket connections
- **AI Integration**: Claude API for assessment, feedback, and AI player generation
- **Matchmaking Engine**: Skill-based party formation with automated AI player inclusion
- **Database**: PostgreSQL with session recordings and portfolio storage

## Design Philosophy

The platform integrates Learning Science principles with balanced motivation approaches. Cognitive Load Theory guides scaffolding, retrieval practice ensures retention, metacognitive instruction develops self-regulation, and formative assessment drives learning. The writer character/avatar system provides visual growth representation, embodying growth mindset through tangible character evolution from seedling to legendary writer.

## Learning Science Foundation

- **Cognitive Load Theory**: Adaptive scaffolding manages working memory constraints
- **Retrieval Practice & Spacing**: Distributed learning builds durable knowledge
- **Metacognitive Development**: Self-regulation strategies for planning, monitoring, evaluation
- **Formative Assessment**: Diagnostic feedback guides learning trajectories
- **Growth Mindset**: Character progression embodies skill development through effort
- **Automaticity Building**: Fluency development frees cognitive resources
- **Balanced Motivation**: Intrinsic support with strategic extrinsic incentives
